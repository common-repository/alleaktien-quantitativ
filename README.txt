== Description ==

Mit einem Shortcode kannst du einzelne Charts von [Eulerpool.com](https://www.eulerpool.com) kostenlos einbetten. Aktienkurs, Umsatz, Dividenden, Dividendenrenditen, Margen, Ausschüttungsquote, Bewertung und viele weitere Charts (siehe unten). Die Einbettung ist kostenlos. Das Copyright der Grafiken und Daten liegt bei Eulerpool Research Systems.

= Eulerpool.com = 
* Tagesaktueller Kurschart für 15.000 Aktien weltweit
* Interaktive Charts für deine Besucher
* 20 Jahre fundamentale Kennzahlen
* Aktien aus USA, Deutschland, Schweiz, Österreich und 30 weiteren Ländern
* kostenlos
* mobil optimiert

= Beispiele =
* `[aaq isin="US4781601046" chart="RevenueChart"]`
* `[aaq isin="US4781601046" chart="DividendChart"]`

= Demo =
Auf [privatanlegerforum.de](https://www.privatanlegerforum.de/eulerpool-das-beste-tool-um-aktien-zu-finden-und-zu-analysieren/ haben wir einige Eulerpool Charts als Demo eingebunden.

= Parameter =
1. **isin** (required): Die ISIN der Aktie
2. **chart** (required): Der Chart-Typ


= Parameter: chart / Chart-Typen = 
1. **QuoteChart**: täglicher Aktienkurs der letzten 20 Jahre
2. **RevenueChart**: Umsatz, EBIT, und Gewinn
3. **RevenuePerShareChart**: Umsatz, EBIT, Gewinn pro Aktie
4. **MarginsChart**: Bruttomarge, EBIT-Marge, Gewinnmarge
5. **DividendChart**: Dividenden
6. **DividendYieldChart**: Dividendenrendite
7. **PayoutRatioChart**: Ausschüttungsquote (Dividenden / Gewinn pro Aktie)
8. **SharesOutstandingChart**: Anzahl ausstehender Aktien
9. **ValuationChart**: KGV, KUV, und EBIT-Multiple


== Installation ==
1. Plugin installieren
2. Plugin aktivieren
3. Shortcode `[aaq isin="..." chart="..."]` in Beiträgen und Seiten verwenden. Die Daten werden automatisch aktualisiert.
